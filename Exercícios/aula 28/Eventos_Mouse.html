Eventos com Mouse


Onmouseclick

Chama uma fução quando clicado.


Onmousedblcick

Chama uma função quando é clicado duas vezes.


Onmouseover

Quando o mouse passa por cima de uma imagem, esta função é ativada.


Onmouseout

Chama a função quando o mouse move para fora de uma imagem


Onmouseup

Quando soltamos o ponteiro do mouse em cima de um parágrafo, esta função é chamada.
